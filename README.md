# Loopis-Project

Projeto Desafio Loopis.

Sistema feito em HTML5 e JavaScript para gerenciar Alunos, turmas e professores.

O Sistema possiu uma pagina inicial com um MENU contendo as opções(ALUNOS, PROFESSORES, TURMAS).

Após escolher qual gerenciar, segue o preenchimento dos campos relacionados a cada um dos mesmos.

Os dados preenchidos apos cadastrados são listados em uma tabela abaixo dos campos do furmlário. 

O desenho do "Lápis" corresponde a operação de edição dos dados.
O desenho do "X" corresponde a operação de exclussão dos dados.
